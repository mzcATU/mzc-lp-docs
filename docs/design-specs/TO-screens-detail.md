## 1. 대시보드 (Dashboard)

### 화면 개요

| 항목 | 내용 |
|------|------|
| 화면 ID | `dashboard` |
| 목적 | 운영 현황 요약 및 핵심 지표 모니터링 |

### UI 구성

```
┌─────────────┬─────────────┬─────────────┬──────────────┐
│ 총 매출     │ 활성 수강생  │ 총 강의 수   │ 신규 가입     │
│ ₩2.4억     │ 1,234명     │ 56개        │ 89명         │
│ ↑15%       │ ↑8%        │ ↑3%         │ ↑25%        │
├─────────────┴─────────────┴─────────────┴──────────────┤
│ ┌─────────────────────────┐ ┌─────────────────────────┐│
│ │    월별 매출 (Area)      │ │   카테고리별 (Pie)      ││
│ └─────────────────────────┘ └─────────────────────────┘│
├─────────────────────────────────────────────────────────┤
│ 인기 강의 (테이블)                                       │
└─────────────────────────────────────────────────────────┘
```

### 통계 카드

| 카드명 | 아이콘 | 데이터 |
|--------|--------|--------|
| 총 매출 | DollarSign | 금액 + 변화율 |
| 활성 수강생 | UserCheck | 인원 + 변화율 |
| 총 강의 수 | BookOpen | 개수 + 변화율 |
| 신규 가입 | TrendingUp | 인원 + 변화율 |

### 차트

- **월별 매출**: Recharts AreaChart, 그라데이션 `#6778ff`
- **카테고리별 비중**: Recharts PieChart

## 2. 강의 목록 (Courses)

### 화면 개요

| 항목 | 내용 |
|------|------|
| 화면 ID | `courses` |
| 목적 | 전체 강의 조회, 검색, 신규 등록 |

### UI 구성

```
┌─────────────────────────────────────────────────────────┐
│ [🔍 검색창]                    [필터] [+ 새 강의 등록]    │
├─────────────────────────────────────────────────────────┤
│ │ [썸네일] 강의명    │ 강사  │ 가격  │수강생│평점│상태│⋮ │
│ │ [img] React 완벽  │ 김개발│₩129K │234 │4.8│활성│⋮ │
├─────────────────────────────────────────────────────────┤
│ [페이지네이션: ◀ 1 2 3 4 5 ▶]                           │
└─────────────────────────────────────────────────────────┘
```

### 테이블 컬럼

| 컬럼 | 타입 | 설명 |
|------|------|------|
| 썸네일 | Image | 60x40 비율 |
| 강의명 | Text | 클릭 시 상세 |
| 강사 | Text | 강사명 |
| 가격 | Number | ₩ 포맷 |
| 수강생 | Number | 인원 수 |
| 평점 | Number | 0.0~5.0 |
| 상태 | Badge | 활성/베스트/임시저장 |
| 액션 | Dropdown | 수정/통계/삭제 |

### 상태 뱃지

| 상태 | 스타일 |
|------|--------|
| 활성 | `bg-brand-green/15 text-brand-green` |
| Best Seller | `bg-brand-purple/25 text-brand-purple` |
| 임시저장 | `bg-gray-500 text-white` |

### 강의 등록 모달

| 필드 | 타입 | 필수 |
|------|------|------|
| 썸네일 | File Upload | X |
| 강의명 | Text | O |
| 강사명 | Text | O |
| 카테고리 | Select | O |
| 가격 | Number | O |
| 상태 | Select | X |
| 설명 | Textarea | X |

## 3. 차수 관리 (Sessions)

### 화면 개요

| 항목 | 내용 |
|------|------|
| 화면 ID | `sessions` |
| 목적 | 강의 차수(기수) 생성 및 관리 |

### 테이블 컬럼

| 컬럼 | 설명 |
|------|------|
| 차수 | 1기, 2기 등 |
| 강의명 | 연결된 강의 |
| 시작일 | YYYY-MM-DD |
| 종료일 | YYYY-MM-DD |
| 수강생 | 등록된 수강생 수 |
| 상태 | 모집중/진행중/종료 |
| 액션 | 수정/수강생보기/삭제 |

### 상태 뱃지

| 상태 | 스타일 |
|------|--------|
| 모집중 | `bg-brand-purple/25 text-brand-purple` |
| 진행중 | `bg-brand-green/15 text-brand-green` |
| 종료 | `bg-gray-500 text-white` |

### 차수 등록 모달

| 필드 | 타입 | 필수 |
|------|------|------|
| 강의 선택 | Select | O |
| 차수 | Text | O |
| 시작일 | DatePicker | O |
| 종료일 | DatePicker | O |
| 최대 수강생 | Number | X |
| 상태 | Select | X |

## 공통 컴포넌트

### Header

| 요소 | 설명 |
|------|------|
| 로고 | "MZRUN" 그라데이션 |
| 검색창 | 플레이스홀더: "검색..." |
| 알림 | Bell 아이콘 |
| 프로필 | 아바타 이미지 |

### Sidebar

| 요소 | 설명 |
|------|------|
| 네비 아이템 | 아이콘 + 텍스트 |
| 확장 메뉴 | ChevronDown/Right |
| 선택 상태 | 배경색 + 좌측 보더 |

### Dialog/Modal

```tsx
<Dialog>
  <DialogTrigger>트리거</DialogTrigger>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>제목</DialogTitle>
    </DialogHeader>
    {/* 내용 */}
    <DialogFooter>
      <Button variant="outline">취소</Button>
      <Button>확인</Button>
    </DialogFooter>
  </DialogContent>
</Dialog>
```

## 인터랙션

### 네비게이션

| 액션 | 결과 |
|------|------|
| 메뉴 클릭 | 해당 화면 전환 |
| 서브메뉴 토글 | 펼침/접힘 |
| 테마 전환 | 300ms 트랜지션 |

### 테이블

| 액션 | 결과 |
|------|------|
| 검색 | 필터링 |
| 행 액션 클릭 | 드롭다운 메뉴 |
| 페이지 이동 | 데이터 로드 |

### 모달

| 액션 | 결과 |
|------|------|
| 열기 | 오버레이 + 모달 |
| 닫기 | X/오버레이/ESC |
| 제출 | 데이터 저장 |
