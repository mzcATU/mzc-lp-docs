# 04. UX Pages

> 페이지별 UX 가이드 - 와이어프레임, 사용자 플로우

---

## 핵심 규칙

```
✅ 역할별 레이아웃 일관성 유지 (SA/TA/TO/TU)
✅ 공통 페이지 패턴 준수 (대시보드, 목록, 상세, 폼)
✅ 페이지 전환 시 브레드크럼 표시
✅ 사용자 플로우 3단계 이내 유지
```

---

## 1. 페이지 UX 개요

### 1.1 역할별 페이지 맵

```
┌─────────────────────────────────────────────────────────────┐
│  SA (System Admin)                                          │
│  ├── Dashboard: 시스템 전체 통계                             │
│  ├── Tenants: 테넌트 목록/상세/생성                          │
│  └── Settings: 시스템 설정                                   │
├─────────────────────────────────────────────────────────────┤
│  TA (Tenant Admin)                                          │
│  ├── Dashboard: 테넌트 통계                                  │
│  ├── Users: 사용자 관리                                      │
│  └── Settings: 테넌트 설정                                   │
├─────────────────────────────────────────────────────────────┤
│  TO (Tenant Operator)                                       │
│  ├── Dashboard: 운영 현황                                    │
│  ├── Contents: 콘텐츠 관리                                   │
│  └── Learning: 학습 관리                                     │
├─────────────────────────────────────────────────────────────┤
│  TU (Tenant User)                                           │
│  ├── Dashboard: 개인 현황                                    │
│  ├── My Learning: 내 학습                                    │
│  └── Teaching: 강의/콘텐츠 관리 (강사)                        │
└─────────────────────────────────────────────────────────────┘
```

## 2. 공통 페이지 패턴

### 2.1 대시보드 페이지 UX

#### 와이어프레임

```
┌─────────────────────────────────────────────────────────────┐
│  대시보드                                                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐    │
│  │  총 강의  │  │ 진행 중  │  │  완료    │  │  수강생  │    │
│  │   24     │  │   12     │  │   8      │  │   156    │    │
│  └──────────┘  └──────────┘  └──────────┘  └──────────┘    │
│                                                             │
│  ┌────────────────────────────┐  ┌────────────────────────┐ │
│  │  최근 활동                  │  │  알림                  │ │
│  │  ─────────────────────────│  │  ─────────────────────│ │
│  │  • 새 강의 등록 (2시간 전)  │  │  • 시스템 점검 안내    │ │
│  │  • 수강생 3명 추가 (4시간)  │  │  • 신규 기능 출시      │ │
│  │  • 콘텐츠 업로드 (어제)     │  │  • 업데이트 예정       │ │
│  └────────────────────────────┘  └────────────────────────┘ │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### UX 가이드라인

| 요소 | 가이드 |
|------|--------|
| 통계 카드 | 핵심 지표 4개 이내, 클릭 시 상세 페이지 이동 |
| 최근 활동 | 최신순 5-10개, 더보기 링크 |
| 알림 | 읽지 않은 항목 강조 |
| 새로고침 | 자동 새로고침 또는 수동 버튼 |

### 2.2 목록 페이지 UX

#### 와이어프레임

```
┌─────────────────────────────────────────────────────────────┐
│  강의 관리                                     [ + 새 강의 ]│
├─────────────────────────────────────────────────────────────┤
│  ┌──────────────────────────────────────┐  [상태 ▼] [정렬 ▼]│
│  │  🔍 강의명으로 검색...               │                   │
│  └──────────────────────────────────────┘                   │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────────┐│
│  │  □  강의명           │ 차시  │ 수강생 │ 상태   │ 생성일  ││
│  ├─────────────────────────────────────────────────────────┤│
│  │  □  React 기초       │  12   │   45   │ 진행중 │ 12/01  ││
│  │  □  TypeScript 심화  │   8   │   32   │ 준비중 │ 12/05  ││
│  │  □  Next.js 입문     │  15   │   67   │ 완료   │ 11/20  ││
│  └─────────────────────────────────────────────────────────┘│
│                                                             │
│                        [ < 1 2 3 4 5 > ]                    │
└─────────────────────────────────────────────────────────────┘
```

#### UX 가이드라인

| 요소 | 가이드 |
|------|--------|
| 검색 | 실시간 검색 또는 Enter 키 제출 |
| 필터 | 자주 사용하는 필터 2-3개 노출 |
| 정렬 | 기본 정렬 + 추가 옵션 드롭다운 |
| 테이블 | 핵심 컬럼 5-7개, 반응형 시 카드로 전환 |
| 페이지네이션 | 10-20개 단위, 또는 무한 스크롤 |
| 빈 상태 | 검색 결과 없음 vs 목록 비어있음 구분 |

### 2.3 상세 페이지 UX

#### 와이어프레임

```
┌─────────────────────────────────────────────────────────────┐
│  < 목록으로                                                 │
│  ─────────────────────────────────────────────────────────│
│  React 기초 강의                           [수정] [삭제]    │
│  진행중 │ 12차시 │ 수강생 45명                              │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────────┐│
│  │  [기본 정보]  [커리큘럼]  [수강생]  [통계]               ││
│  ├─────────────────────────────────────────────────────────┤│
│  │                                                         ││
│  │  강의 설명                                              ││
│  │  ──────────────────────────────────────────────────   ││
│  │  React의 기초를 배우는 강의입니다.                      ││
│  │                                                         ││
│  │  강사 정보                                              ││
│  │  ──────────────────────────────────────────────────   ││
│  │  [👤] 홍길동 │ react@example.com                        ││
│  │                                                         ││
│  └─────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────┘
```

#### UX 가이드라인

| 요소 | 가이드 |
|------|--------|
| 뒤로가기 | 좌측 상단 "< 목록으로" 링크 |
| 제목 영역 | 제목 + 상태 뱃지 + 핵심 정보 요약 |
| 액션 버튼 | 우측 상단 수정/삭제 (삭제는 확인 필요) |
| 탭 | 정보가 많을 경우 탭으로 분리 |
| 편집 | 인라인 편집 또는 수정 페이지 이동 |

### 2.4 생성/수정 페이지 UX

#### 와이어프레임

```
┌─────────────────────────────────────────────────────────────┐
│  < 취소                                                     │
│  ─────────────────────────────────────────────────────────│
│  새 강의 만들기                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  강의명 *                                                   │
│  ┌─────────────────────────────────────────────────────────┐│
│  │  강의 이름을 입력하세요                                  ││
│  └─────────────────────────────────────────────────────────┘│
│                                                             │
│  강의 설명                                                  │
│  ┌─────────────────────────────────────────────────────────┐│
│  │                                                         ││
│  │                                                         ││
│  └─────────────────────────────────────────────────────────┘│
│                                                             │
│  강사 선택                                                  │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  강사를 선택하세요                              ▼   │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                              [ 취소 ]  [ 저장 ]             │
└─────────────────────────────────────────────────────────────┘
```

#### UX 가이드라인

| 요소 | 가이드 |
|------|--------|
| 필수 필드 | "*" 표시 + 라벨 옆에 표시 |
| 유효성 검사 | 실시간 또는 blur 시 검사 |
| 에러 메시지 | 필드 아래 인라인 표시 |
| 취소 | 변경사항 있으면 확인 다이얼로그 |
| 저장 | 제출 중 버튼 비활성화 + 로딩 표시 |
| 성공 | 토스트 + 상세 페이지로 이동 |

## 3. 사용자 플로우 예시

### 3.1 강의 생성 플로우 (TU)

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  강의 목록    │ →   │  + 새 강의   │ →   │  강의 정보   │
│  페이지      │     │  버튼 클릭   │     │  입력       │
└──────────────┘     └──────────────┘     └──────────────┘
                                                ↓
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  강의 상세   │ ←   │  성공 토스트  │ ←   │  저장 버튼   │
│  페이지      │     │  표시        │     │  클릭       │
└──────────────┘     └──────────────┘     └──────────────┘
```

#### 상세 플로우

```
[강의 목록 페이지]
    │
    ▼ "+ 새 강의" 버튼 클릭
    │
[강의 생성 폼]
    │
    ├─ 강의명 입력 (필수)
    ├─ 강의 설명 입력
    └─ 강사 선택
    │
    ▼ "저장" 버튼 클릭
    │
    ├─ [유효성 검사 실패] → 에러 메시지 표시 → 폼 유지
    │
    └─ [유효성 검사 성공] → API 호출
        │
        ├─ [API 실패] → 에러 토스트 → 폼 유지
        │
        └─ [API 성공] → 성공 토스트 → [강의 상세 페이지]
```

---

### 3.2 콘텐츠 업로드 플로우 (TU)

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  콘텐츠 풀   │ →   │  폴더 선택   │ →   │  파일 드롭   │
│  페이지      │     │  (선택)      │     │  또는 선택   │
└──────────────┘     └──────────────┘     └──────────────┘
                                                ↓
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  업로드 완료 │ ←   │  진행률 표시  │ ←   │  업로드 시작 │
│  목록 갱신   │     │  (Progress)  │     │  (자동)      │
└──────────────┘     └──────────────┘     └──────────────┘
```

#### 상세 플로우

```
[콘텐츠 풀 페이지]
    │
    ├─ 좌측: 폴더 트리
    └─ 우측: 콘텐츠 목록
    │
    ▼ 폴더 선택 (선택사항)
    │
    ▼ "파일 업로드" 버튼 클릭 또는 드래그앤드롭
    │
[업로드 시작]
    │
    ├─ 파일 유형 검사
    │   └─ [지원하지 않는 형식] → 에러 메시지
    │
    ├─ 파일 크기 검사
    │   └─ [크기 초과] → 에러 메시지
    │
    └─ [검사 통과] → 업로드 시작
        │
        ▼ Progress Bar 표시 (0% → 100%)
        │
        ├─ [업로드 실패] → 에러 토스트 + 재시도 옵션
        │
        └─ [업로드 성공] → 성공 표시 → 목록 자동 갱신
```

## 4. 페이지 추가 가이드

```markdown

#### 와이어프레임

[ASCII 와이어프레임]
\`\`\`

#### UX 가이드라인

| 요소 | 가이드 |
|------|--------|
| ... | ... |

#### 사용자 플로우 (필요시)

\`\`\`
[플로우 다이어그램]
\`\`\`
```

### 페이지 추가 체크리스트

- [ ] 역할 확인 (SA/TA/TO/TU 중 어디?)
- [ ] 페이지 유형 확인 (대시보드/목록/상세/폼)
- [ ] 공통 레이아웃 적용
- [ ] 브레드크럼 경로 설정
- [ ] 상태 처리 (로딩/에러/빈 상태)
- [ ] 사용자 플로우 3단계 이내 확인
