# 05. UX Components

> 컴포넌트별 UX 가이드 - 상태, 인터랙션, 와이어프레임

---

## 핵심 규칙

```
✅ 모든 인터랙티브 요소에 상태 피드백 제공 (hover, focus, active, disabled)
✅ 키보드 접근성 보장 (Tab, Enter, Escape)
✅ 로딩/에러/성공 상태 처리
✅ 일관된 스타일과 동작 유지
```

---

## 1. 컴포넌트 UX 개요

### 1.1 컴포넌트 분류표

| 분류 | 컴포넌트 | 용도 |
|------|----------|------|
| **기본** | Button, Input, Select, Checkbox | 폼 요소 |
| **피드백** | Toast, Modal, Dialog, Tooltip | 사용자 알림 |
| **네비게이션** | Tabs, Breadcrumb, Sidebar | 페이지 이동 |
| **데이터 표시** | Table, Card, List, Badge | 정보 표시 |
| **복합** | TreeView, FolderTree, FileDropzone | 특수 기능 |

## 2. 기본 컴포넌트 UX

### 2.1 Button

#### 상태별 와이어프레임

```
┌─────────────────────────────────────────────────────────────┐
│  [Default]     [Hover]       [Active]      [Disabled]       │
│  ┌────────┐   ┌────────┐   ┌────────┐    ┌────────┐        │
│  │  저장  │   │  저장  │   │  저장  │    │  저장  │        │
│  └────────┘   └────────┘   └────────┘    └────────┘        │
│   bg-brand    bg-brand     bg-brand      bg-brand          │
│               -hover       -active       opacity-50        │
│               (어두움)      (더 어두움)    cursor-not       │
└─────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────┐
│  [Loading State]                                            │
│                                                             │
│  ┌────────────────┐                                        │
│  │  ◌ 저장 중...  │  ← Spinner + 텍스트 변경               │
│  └────────────────┘    버튼 비활성화                        │
└─────────────────────────────────────────────────────────────┘
```

#### UX 가이드라인

| 상태 | 시각적 변화 | 접근성 |
|------|------------|--------|
| Default | 기본 배경색 | - |
| Hover | 배경 어두워짐 | - |
| Focus | outline ring 표시 | Tab 이동 시 |
| Active | 배경 더 어두워짐 | 클릭 순간 |
| Disabled | 투명도 50%, cursor 변경 | aria-disabled |
| Loading | Spinner 표시, 비활성화 | aria-busy |

#### 버튼 배치 규칙

```
┌─────────────────────────────────────────────────────────────┐
│  [폼 하단]                                                  │
│                                                             │
│                              [ 취소 ]  [ 저장 ]             │
│                              (Ghost)   (Brand)              │
│                              좌측      우측 (Primary)        │
└─────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────┐
│  [목록 헤더]                                                │
│                                                             │
│  강의 관리                                     [ + 새 강의 ]│
│                                                우측 상단    │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 Modal / Dialog

#### 와이어프레임

```
┌─────────────────────────────────────────────────────────────┐
│  ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░│
│  ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░│
│  ░░░░░░░┌─────────────────────────────────────┐░░░░░░░░░░░│
│  ░░░░░░░│  모달 제목                     [X]  │░░░░░░░░░░░│
│  ░░░░░░░├─────────────────────────────────────┤░░░░░░░░░░░│
│  ░░░░░░░│                                     │░░░░░░░░░░░│
│  ░░░░░░░│         모달 콘텐츠 영역            │░░░░░░░░░░░│
│  ░░░░░░░│                                     │░░░░░░░░░░░│
│  ░░░░░░░├─────────────────────────────────────┤░░░░░░░░░░░│
│  ░░░░░░░│            [ 취소 ]  [ 확인 ]       │░░░░░░░░░░░│
│  ░░░░░░░└─────────────────────────────────────┘░░░░░░░░░░░│
│  ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░│
└─────────────────────────────────────────────────────────────┘
  ↑ Backdrop (클릭 시 닫힘 또는 유지)
```

#### UX 가이드라인

| 요소 | 가이드 |
|------|--------|
| 열기 | 페이드 인 애니메이션 (150-200ms) |
| 닫기 | X 버튼, 취소 버튼, Escape 키, Backdrop 클릭 |
| 포커스 | 열릴 때 모달 내부로 포커스 이동 |
| 포커스 트랩 | Tab 이동이 모달 내부에서만 순환 |
| 스크롤 | 모달 열릴 때 body 스크롤 잠금 |
| 버튼 순서 | 좌측: 취소(Ghost), 우측: 확인(Primary) |

#### 확인 다이얼로그 (AlertDialog)

```
┌─────────────────────────────────────────┐
│  삭제 확인                        [X]   │
├─────────────────────────────────────────┤
│                                         │
│  ⚠️ 이 작업은 되돌릴 수 없습니다.       │
│                                         │
│  "React 기초 강의"를 정말                │
│  삭제하시겠습니까?                       │
│                                         │
├─────────────────────────────────────────┤
│              [ 취소 ]  [ 삭제 ]         │
│                        (Danger)         │
└─────────────────────────────────────────┘
```

| 유형 | Primary 버튼 | 색상 |
|------|-------------|------|
| 정보 확인 | 확인 | Brand |
| 삭제 확인 | 삭제 | Danger (빨강) |
| 경고 | 계속 | Warning (주황) |

## 3. 복합 컴포넌트 UX

### 3.1 TreeView (계층 구조)

#### 와이어프레임

```
┌─────────────────────────────────────────────────────────────┐
│  커리큘럼 구성                              [ + 폴더 ] [ + 차시 ]│
├─────────────────────────────────────────────────────────────┤
│  ▼ 📁 1주차: React 기초                          [⋮]        │
│     ├─ 📄 1-1. React 소개 (10분)                 [⋮]        │
│     ├─ 📄 1-2. JSX 이해하기 (15분)               [⋮]        │
│     └─ 📄 1-3. 컴포넌트 기초 (20분)              [⋮]        │
│  ▼ 📁 2주차: 상태 관리                           [⋮]        │
│     ├─ 📄 2-1. useState (12분)                  [⋮]        │
│     └─ 📄 2-2. useEffect (18분)                 [⋮]        │
│  ▶ 📁 3주차: 고급 패턴 (접힘)                    [⋮]        │
└─────────────────────────────────────────────────────────────┘
```

#### 인터랙션 플로우

```
[접기/펼치기]
▼ 폴더 클릭 → ▶ 접힘 (애니메이션)
▶ 폴더 클릭 → ▼ 펼침 (애니메이션)

[드래그앤드롭]
항목 드래그 시작
    ↓
드롭 영역 하이라이트 표시
    ↓
드롭 → 순서 변경 (애니메이션)
    ↓
API 호출 → 저장

[컨텍스트 메뉴]
[⋮] 클릭 → 드롭다운 메뉴
├─ 수정
├─ 복제
├─ 이동
└─ 삭제
```

#### UX 가이드라인

| 기능 | 가이드 |
|------|--------|
| 펼치기/접기 | 화살표 또는 폴더 아이콘 클릭, 부드러운 애니메이션 |
| 드래그앤드롭 | 드래그 중 시각적 피드백, 드롭 가능 영역 표시 |
| 선택 | 단일 선택 하이라이트, 다중 선택 (Ctrl/Cmd + 클릭) |
| 컨텍스트 메뉴 | 더보기 버튼 또는 우클릭 |
| 깊이 표시 | 들여쓰기로 계층 표현 (level × 16px) |

### 3.2 FileDropzone (파일 업로드)

#### 와이어프레임

```
┌─────────────────────────────────────────────────────────────┐
│  [기본 상태]                                                │
│  ┌─────────────────────────────────────────────────────────┐│
│  │                                                         ││
│  │                    📁                                   ││
│  │                                                         ││
│  │         파일을 여기에 드래그하거나                       ││
│  │         [파일 선택] 버튼을 클릭하세요                    ││
│  │                                                         ││
│  │         지원 형식: mp4, pdf, jpg (최대 2GB)             ││
│  │                                                         ││
│  └─────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────┐
│  [드래그 중 상태]                                           │
│  ┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┐│
│  │                                                         ││
│  │                    📥                                   ││
│  │                                                         ││
│  │              여기에 파일을 놓으세요                      ││
│  │                                                         ││
│  └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┘│
│    ↑ 테두리 점선 + 배경색 변경 + 아이콘 변경                │
└─────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────┐
│  [업로드 중 상태]                                           │
│  ┌─────────────────────────────────────────────────────────┐│
│  │  video.mp4                                        [X]   ││
│  │  ████████████████████░░░░░░░░░░░░░░░░  45%  2.3MB/5MB  ││
│  ├─────────────────────────────────────────────────────────┤│
│  │  document.pdf                                     [X]   ││
│  │  ████████████████████████████████████  100%  ✅        ││
│  └─────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────┘
```

#### 업로드 플로우

```
[파일 선택/드롭]
    │
    ▼
[유효성 검사]
    │
    ├─ [파일 형식 오류] → 에러 메시지 "지원하지 않는 형식입니다"
    ├─ [파일 크기 초과] → 에러 메시지 "파일 크기가 2GB를 초과합니다"
    │
    └─ [검사 통과]
        │
        ▼
    [업로드 시작]
        │
        ▼
    [Progress Bar 표시] (0% → 100%)
        │
        ├─ [업로드 실패] → 에러 표시 + 재시도 버튼
        │
        └─ [업로드 성공] → ✅ 체크 표시 + 목록에 추가
```

#### UX 가이드라인

| 상태 | 시각적 피드백 |
|------|-------------|
| 기본 | 점선 테두리, 안내 텍스트 |
| 드래그 중 | 테두리 강조, 배경색 변경, 아이콘 변경 |
| 업로드 중 | Progress Bar, 파일명, 크기, 취소 버튼 |
| 완료 | 체크 아이콘, 녹색 표시 |
| 에러 | 빨간 테두리, 에러 메시지, 재시도 버튼 |

## 4. 컴포넌트 추가 가이드

### 새 컴포넌트 추가 시 템플릿

```markdown
### [컴포넌트명]

#### 와이어프레임

\`\`\`
[상태별 ASCII 와이어프레임]
\`\`\`

#### UX 가이드라인

| 상태/요소 | 가이드 |
|----------|--------|
| ... | ... |

#### 인터랙션 플로우 (필요시)

\`\`\`
[플로우 다이어그램]
\`\`\`
```

### 컴포넌트 추가 체크리스트

- [ ] 상태 정의 (default, hover, focus, active, disabled, loading, error)
- [ ] 키보드 접근성 (Tab, Enter, Escape)
- [ ] 스크린 리더 지원 (aria-label, role)
- [ ] 애니메이션 (열기/닫기, 상태 전환)
- [ ] 반응형 처리 (모바일 대응)
- [ ] 에러 처리 (에러 메시지, 복구 방법)
